# Smart Petrol Cost Calculator with BUDI MADANI Rebate

This is an Android mobile application developed for the ICT602 Mobile Technology assignment.

## Application Purpose

The application helps users estimate petrol cost in Malaysia and calculate the BUDI MADANI rebate for eligible RON95 users.

## Features

- Home screen with navigation options
- Petrol cost calculator screen
- Petrol type selection: RON95, RON97, Diesel
- Petrol price per liter input
- Fuel usage input
- BUDI MADANI eligibility selection
- Total petrol cost calculation
- BUDI rebate calculation for eligible RON95 users
- Final payable amount display
- Total savings display
- About page with application icon, author information, copyright notice, and GitHub repository link
- Top bar menu navigation
- Share function

## Formula Used

Total petrol cost = fuel usage × petrol price per liter

BUDI rebate = fuel usage × RM1.99

The BUDI rebate is only applied when:
- Petrol type is RON95
- BUDI MADANI eligibility is Yes

Final payable amount = total petrol cost - BUDI rebate

Total savings = BUDI rebate

## Sample Calculation

Petrol type: RON95  
Petrol price per liter: RM4.27  
Fuel usage: 40 liters  
BUDI MADANI eligibility: Yes  

Total petrol cost = 40 × RM4.27 = RM170.80  
BUDI rebate = 40 × RM1.99 = RM79.60  
Final payable amount = RM170.80 - RM79.60 = RM91.20
